NetAnim 소스코드 다운로드 및 빌드
# 1. ns3 폴더로 이동
cd ~/cpp_study/ns3/
# 2. NetAnim 코드 다운로드 (Git)
git clone https://gitlab.com/nsnam/netanim.git
# 3. NetAnim 폴더로 이동
cd netanim
# 4. 컴파일 환경 세팅 및 빌드 (약 1~2분 소요됩니다)
mkdir build
cd build
cmake ..
make

실행
./netanim
~/cpp_study/ns3/netanim/build/netanim

차이점 : 원점 (0,0) 위치, Y값이 커지는 방향, 자동차의 움직임 순
SUMO (수학/지도 좌표계)         :맵의 정중앙 (또는 왼쪽 아래), 위쪽 (북쪽, ↑), 남쪽(-100) ➡️ 북쪽(+100)으로 직진
NetAnim / NS-3 (화면 좌표계)    :화면의 왼쪽 맨 위 (Top-Left), 아래쪽 (남쪽, ↓), 위쪽(-100) ➡️ 아래쪽(+100)으로 직진

OMNeT++ IDE에서 Veins 불러오기:
터미널에서 omnetpp
상단 메뉴에서 File > Import 클릭.
General > Existing Projects into Workspace 선택 후 'Next'.
Select root directory에서 아까 압축을 푼 veins-5.2 폴더를 찾아 선택하고 'Finish'.
Veins 프로젝트 빌드:
IDE 왼쪽 패널(Project Explorer)에 veins 폴더가 생겼을 거야.
해당 폴더를 우클릭하고 Build Project를 클릭해.
에러 없이 완료되면 연동 준비 끝
cd ~/veins-veins-5.2
python3 bin/veins_launchd -vv -c sumo
다시 캡처해서 보여준 OMNeT++ IDE 창으로 돌아가.
왼쪽 Project Explorer에서 veins 폴더를 열고, examples 폴더 안에 있는 veins 폴더로 들어가 봐.
거기에 omnetpp.ini 라는 설정 파일이 있을 거야.
그 파일을 우클릭한 다음, Run As > OMNeT++ Simulation을 클릭해
잠시 후 새로운 시각화 창(Qtenv)이 뜰 텐데, 창 왼쪽 상단의 재생(Play) 버튼을 누르면 시뮬레이션이 시작돼.
